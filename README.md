# SCSS Gulp Boilerplate

>Very Basic Sass(SCSS) boilerplate.

*Updated 7/4/2017 - Add sass source maps*


### Prerequisites
* node
* npm
* gulp

## Quick Start:

```shell

# Browse to project folder
cd /path/to/project/this/folder

# Install browserSync and other dependencies
npm install

# Start server
gulp watch

```
